Práctica 1: Arduino Básico

Diego Rodríguez Orozco
Ing. Sistemas Computacionales
Fecha: Agosto 28, 2023
Arduino Electrónica

- Resumen -

El objetivo de esta práctica, es desarrollar sistemas y circuitos de Arduino simples para empezar con las bases de la programación en un microcontrolador, en este caso se implementará en un Arduino Uno.

- Introducción -

Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores. Esta plataforma permite crear diferentes tipos de microordenadores de una sola placa a los que la comunidad de creadores puede darles diferentes tipos de uso.

El hardware libre son los dispositivos cuyas especificaciones y diagramas son de acceso público, de manera que cualquiera puede replicarlos. Esto quiere decir que Arduino ofrece las bases para que cualquier otra persona o empresa pueda crear sus propias placas, pudiendo ser diferentes entre ellas pero igualmente funcionales al partir de la misma base.

- Materiales -

  • Protoboard

  • LED's

  • Push down

  • Protoboard

  • Arduino Uno

  • Display de 7 segmentos

- Desarrollo -

Para esta práctica, el profesor nos brindó una carpeta de Tinkercad con 18 circuitos que debíamos replicar en físico con el protoboard y los componentes electrónicos. El proceso era principalm,ente, comprender el circuito y simularlo, para después armar el circuito y ejecutar el código en el arduino.

Simulación

Se simularon los circuitos en Thinkercad.
