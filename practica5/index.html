Práctica 5: Pantalla OLED

Diego Rodríguez Orozco
Ing. Sistemas Computacionales
Fecha: Octubre 19, 2023
ATTINY Electrónica

- Resumen -

Para esta practica, con el ATTINY, se utilizó un modulo del arduino que es la pantalla OLED, capaz de mostrar información que queramos. En este caso, que cuando se presione un botón, se muestre la panatalla la palabra On y si no se presiona, que se muestre la palabra OFF.

- Introducción -

Las pantallas OLED se componen de píxeles individuales que emiten su propia luz cuando se les aplica una corriente eléctrica. Esto significa que no necesitan retroiluminación, como las pantallas LCD convencionales, lo que las hace más delgadas y eficientes energéticamente. Además, los píxeles en una pantalla OLED pueden encenderse y apagarse de forma independiente, lo que permite lograr negros verdaderos y colores vivos, lo que las hace ideales para la representación de imágenes y texto.

- Materiales -

  • Protoboard

  • Arduino Uno

  • Pantalla OLED

  • Botón push down

- Desarrollo -

Para el desarrollo de esta práctica, fue necesario primero aprender a usar la pantalla por medio de la simulación en WOWKI, donde se aprendió a manipular y mostrar la información con el manejo de sus pixeles de forma que se logró exitosamente seguir con el plan que se estipuló en el objetivo. Ya que se logró finalizar con la simulación, se implemento la pantalla con su conexión como corresponde, y se incluyeron dos botones push down, al presionar uno, indicaria la pantalla la palabra OFF y al presionar el otro, la palabra ON

Simulación

Se simularon los circuitos en WOKWI.

- Resultados -

Después de analizar la simulación, se ejecutó de forma correcta los circuitos con la funcionalidad requerida.

- Conclusiones -

Se logró completar la funcionalidad básica de una pantalla OLED, además de entender las conexiones de la misma y la programación necesaria.
